# MinimalPosteriorSampling
Minimal toy project for score-based posterior sampling based on 

Song, Y., Sohl-Dickstein, S., Kingma, D. P., et al. (2021). 
Score-based generative modeling through stochastic differential equations. ICLR 2021

Daras, G., Chung, H.,  Lai, C.-H., et al. (2024).
A Survey on Diffusion Models for Inverse Problems. arxiv:2410.00083
